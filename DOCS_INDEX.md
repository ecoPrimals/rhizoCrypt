## 🎓 Key Concepts

**Vertex**: Content-addressed event (Blake3)  
**Session**: Scoped DAG with lifecycle  
**Slice**: Checkout from permanent storage  
**Dehydration**: Commit session to LoamSpine  
**Capability**: Service discovery by function, not name  
**Pure Infant**: Zero hardcoded primal knowledge
