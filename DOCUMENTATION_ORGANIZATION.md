## 🔄 Document Lifecycle

### Active → Archive
Documents move to archive when:
1. They're session-specific (dated)
2. They're superseded by newer versions
3. They're detailed analysis (vs executive summary)
4. They're useful for historical reference only

### Active Documents
Must be:
1. Always current
2. Frequently referenced
3. Essential for understanding
4. Quick reference material
